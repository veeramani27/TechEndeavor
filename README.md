# Application Description

Project Name: TechEndeavor
Stack: FastAPI (backend) + React with Vite (frontend) + PostgreSQL (Neon)
ORM : SQL Alchemy
Version: 1.0.0

## Overview:
This is a full-stack web application combining a FastAPI backend with a React frontend powered by Vite. The backend handles API requests, data processing, and business logic, while the frontend provides a responsive, modern UI that communicates with the backend via RESTful APIs.

## Workflow of the application

- The Authenticated user can add the blog 
- Anyone (even guest) can view and read the blog

## FastAPI Backend (Folder : backend):

- Lightweight, asynchronous, and highly performant APIs.
- Database integration PostgreSQL using neon.
- Secure authentication and user management (initially one user (myself)).
